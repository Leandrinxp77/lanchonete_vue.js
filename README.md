🍔 Controle de Pedidos — Vue.js via CDN

Aplicação desenvolvida com Vue.js via CDN para controle de pedidos de uma lanchonete, utilizando os recursos de Computed Properties e Watchers.

📌 Objetivo

O projeto foi criado com foco em demonstrar:

Reatividade com Vue.js
Uso de computed
Uso de watch
Manipulação de listas e formulários com v-model
Atualização automática da interface
🚀 Funcionalidades
✅ Lista de produtos

Cada produto possui:

Nome
Preço
Quantidade
✅ Computed Properties

A aplicação calcula automaticamente:

Subtotal do pedido
Taxa de entrega
Desconto
Valor total
Quantidade total de itens
✅ Watchers

A aplicação monitora alterações e executa ações automaticamente:

Exibe mensagem quando o subtotal ultrapassa R$ 100
Exibe aviso quando a quantidade de refrigerantes é maior que 5
Registra atualizações do pedido no console
✅ Extras implementados
Tema escuro
Botão para limpar pedido
Interface estilizada
Layout responsivo
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript
Vue.js 3 via CDN
